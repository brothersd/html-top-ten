10 HTML Tags Examples

1. <h1> - Heading

This is the main heading of the page.

<h1>This is a main heading</h1>

2. <p> - Paragraph

This is a paragraph element that contains blocks of text content.

<p>This is a paragraph element</p>

3. <a> - Anchor/Link

Visit Example Website for more information.

<a href="https://www.example.com">Example Website</a>

4. <img> - Image

An image displayed on the webpage.

<img src="image.jpg" alt="Description" width="150">

5. <ul>, <ol>, <li> - Lists

Unordered List:

  • Apples
  • Bananas
  • Oranges

Ordered List:

  1. First step
  2. Second step
  3. Third step
<ul><li>Item</li></ul>
<ol><li>Item</li></ol>

6. <div> - Division/Container

This is a div container used to group other elements together.
<div>Content inside a container</div>

7. <span> - Inline Container

This text contains a highlighted span element.

<span style="background-color: yellow;">highlighted text</span>

8. <button> - Button

A clickable button element.

<button>Click Me</button>

9. <form> and <input> - Form Elements




<form><input type="text" placeholder="Name"></form>

10. <strong> and <em> - Text Formatting

This text is strong/bold and this is emphasized/italic.

<strong>bold text</strong>
<em>italic text</em>
